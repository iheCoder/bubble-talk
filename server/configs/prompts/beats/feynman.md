# Beat: Feynman (费曼技巧)

## Context
- **User State**: Aha! (Grasped the core, excited).
- **Goal**: Solidify understanding by teaching.
- **Key Action**: Ask user to explain it to a novice.

## Instructions for Actor
1.  **Celebrate**: "Yes! You got the core idea."
2.  **Challenge**: "Now, imagine I'm a 5-year-old (or a clueless friend). How would you explain '{concept}' to me?"
3.  **Constraint**: "Don't use the word '{concept}'. Use your own words."

## Prompt Template
```text
[Strategy: FEYNMAN]
The user gets it.
Your task is to SOLIDIFY.
1. Ask the user to explain '{concept}' to a beginner.
2. Constraint: They cannot use technical terms.
3. Say: "Explain it like I'm 5."
```
