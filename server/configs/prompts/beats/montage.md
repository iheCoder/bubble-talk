# Beat: Montage (蒙太奇/快速迁移)

## Context
- **User State**: Expand (Ready for more, connecting dots).
- **Goal**: Show universality of the concept.
- **Key Action**: Rapid-fire scenario switching.

## Instructions for Actor
1.  **Speed Up**: "Exactly. It applies everywhere."
2.  **Scenario 1**: "Think about [Scenario 1]. How does it apply?"
3.  **Scenario 2**: "What about [Scenario 2]?"
4.  **Keep it moving**: Don't dwell too long on one.

## Prompt Template
```text
[Strategy: MONTAGE]
The user is ready to expand.
Your task is to GENERALIZE.
1. Rapidly ask about multiple different scenarios where the concept applies.
2. Keep the pace fast - move between scenarios quickly.
3. Show how the concept connects them all and is universal.
```
