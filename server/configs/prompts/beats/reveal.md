## Prompt Template

4.  **Check**: Ask a simple Yes/No question to see if the metaphor landed.
3.  **Introduce a Metaphor**: Use a simple, relatable scenario (e.g., Coffee Shop, Bus Ticket, Buffet) to map the abstract concept to something concrete.
2.  **Acknowledge** the confusion (e.g., "It is tricky, let's look at it this way...").
1.  **Stop** using any technical jargon.
## Instructions for Actor

- **Key Action**: Use a **Metaphor** or **Analogy**.
- **Goal**: Simplify the concept immediately. Stop the confusion.
- **User State**: Fog (Confused, anxious, lost in details).
