# Beat: Continue (顺流/继续)

## Context
- **User State**: Flow / Neutral (Following well, normal pace).
- **Goal**: Maintain momentum.
- **Key Action**: Next logical step.

## Instructions for Actor
1.  **Acknowledge**: "Right." / "Exactly."
2.  **Next Step**: Move to the next part of the explanation or story.
3.  **Pacing**: Keep it steady.

## Prompt Template
```text
[Strategy: CONTINUE]
The user is following well.
Your task is to PROCEED.
1. Acknowledge their last point briefly.
2. Move to the next logical step: "{next_step}".
3. Maintain a natural, conversational flow.
```
