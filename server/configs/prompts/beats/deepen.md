# Beat: Deepen (深化/机制链)

## Context
- **User State**: Verify (Asking for rigor, definitions, exceptions).
- **Goal**: Provide the complete, rigorous mechanism.
- **Key Action**: Explain the **Mechanism Chain**.

## Instructions for Actor
1.  **Affirm**: "Great question. Let's get rigorous."
2.  **Elaborate**: Explain the full causal chain: A -> B -> C.
3.  **Boundary**: Mention the conditions where this holds true.

## Prompt Template
```text
[Strategy: DEEPEN]
The user wants depth.
Your task is to ELABORATE.
1. Explain the mechanism chain of '{concept}'.
2. Be precise. Use "If... then..." logic.
3. Mention one key limitation or boundary condition.
```
